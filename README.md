# git-stage-batch

Non-interactive hunk-by-hunk and line-by-line staging for git, similar to `git add -p` but designed for automation.

```
   ┌─────────────────┐
   │ git-stage-batch │
   └─────────────────┘

       o───o
      /
  o───o───o

 stage patches in batches
```
